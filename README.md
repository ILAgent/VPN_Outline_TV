# Native Outline VPN Client for Android, Android TV, and Google TV

This is a native, open-source, lightweight VPN application designed for devices running Android, Android TV, and Google TV.

### Features

*   Unlike the original version written in Apache Cordova, this is a native Android application: it has a smaller footprint and operates more stably.
*   **Whitelist:** Use the VPN only for selected applications without needing to constantly turn it on and off.
*   **Multi-Platform Support:** Compatible with devices running Android, Android TV, and Google TV.
*   **Secure Connections:** Encrypt your data and maintain privacy with robust security measures.
*   **User-Friendly Interface:** Simple and intuitive design for easy navigation and use across all supported platforms.

# Нативный Outline VPN Клиент для Android, Android TV и Google TV

Это нативное open-source, легковесное VPN-приложение, разработанное для устройств на базе Android, Android TV и Google TV.

### Особенности

*   В отличие от оригинальной версии, написанной на apache cordova, это нативное андроид приложение: имеет меньший размер, стабильнее работает.
*   **Белый список:** используйте vpn только для избранных приложений без необходимость постоянного включения и отключения.
*   **Поддержка нескольких платформ:** Совместимо с устройствами на базе Android, Android TV и Google TV.
*   **Безопасные подключения:** Шифруйте свои данные и сохраняйте конфиденциальность с помощью надежных средств безопасности.
*   **Удобный интерфейс: Простой** и интуитивно понятный дизайн для легкой навигации и использования на всех поддерживаемых платформах.
